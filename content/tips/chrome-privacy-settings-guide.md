大部分 Chrome 用户从未改过隐私设置。Google 默认的设置对广告商有利，对你不利——它会收集你的浏览数据来投放精准广告。

这篇文章列出了 Chrome 中 **9 个最应该修改的隐私设置**。不是让你改所有设置（那样太影响体验），而是帮你找出那些"不改有风险，改了对体验几乎没影响"的关键选项。

按重要性排序，越靠前越应该改。

## 设置一：关闭第三方 Cookie（风险等级：高）

**不改会怎样：** 第三方 Cookie 让广告商在你访问的每个网站上追踪你。你在淘宝搜索了"运动鞋"，之后在任何网站都会看到运动鞋广告——这就是第三方 Cookie 干的事。

**操作：** chrome://settings/cookies → "第三方 Cookie" → 选择"仅限隐身模式"

**为什么只选"仅限隐身模式"而不是"阻止所有"：** 选择"阻止所有"会导致很多网站的登录功能失效（比如无法保持登录状态）。选择"仅限隐身模式"意味着正常浏览时第三方 Cookie 仍然工作（网站功能正常），但隐身模式下会自动阻止（隐私浏览时不被追踪）。

## 设置二：禁止网站追踪你的物理位置（风险等级：高）

![#设置二：禁止网站追踪你的物理位置（风险](/images/tips/chrome-privacy-settings-guide/body3.jpg)

**不改会怎样：** 很多网站会请求获取你的地理位置，用于投放本地广告或推送附近商家信息。Chrome 默认会弹出位置请求，但用户经常不小心点了"允许"。

**操作：** chrome://settings/privacy → "网站权限" → "位置信息" → 设为"不允许网站查看你的位置信息"

**如果你确实需要给地图网站授权位置：** 可以在需要时再单独允许，而不是全局开启。在"位置信息"页面点击"添加"按钮，输入允许的网站地址。

## 设置三：管理网站发送通知的权限（风险等级：中）

**不改会怎样：** 随着你授权通知的网站越来越多，浏览器右下角会不断弹出通知，影响使用体验。更糟糕的是，某些恶意网站会利用通知权限推送钓鱼广告。

**操作：** chrome://settings/notifications → 查看已授权通知的网站列表 → 移除不需要的通知权限

**建议：** 只保留你确实需要的通知（如 Gmail、日历），移除所有新闻、电商、社交媒体的通知权限。

## 设置四：清除浏览数据（风险等级：中）

![#设置四：清除浏览数据（风险等级：中）](/images/tips/chrome-privacy-settings-guide/body2.jpg)

**不改会怎样：** Chrome 会保存你所有的浏览记录、下载历史、密码和表单数据。如果电脑被别人使用或中了恶意软件，这些数据都可能被窃取。

**操作：** Chrome → 设置 → 隐私和安全 → 清除浏览数据 → 勾选需要清除的内容 → 设置时间范围

**建议的时间范围：**

- 偶尔清理：选择"过去 4 周"
- 定期清理：选择"过去 24 小时"或"过去 7 天"
- 浏览敏感内容后：选择"过去 1 小时"

**注意：** 清除浏览数据时不要勾选"密码和其他登录数据"（除非你有密码管理器备份），否则所有已保存的登录信息都会丢失。手动+自动清理的完整方法参考 [Chrome 缓存清理教程](/tips/chrome-clear-cache/)。

## 设置五：启用安全浏览（风险等级：高）

**这个设置 Chrome 默认已开启，但请确认一下。**

**操作：** chrome://settings/security → 确保"安全浏览"已开启，选择"保护"

安全浏览会检查你访问的每个网站，如果网站被标记为钓鱼或恶意网站，Chrome 会显示红色警告页面。这是 Chrome 最基础的安全保护功能，不要关闭。

**如果安全浏览被关闭了：** 你可能在不自知的情况下访问钓鱼网站，输入的账号密码会被窃取。

## 设置六：检查网站权限（摄像头、麦克风、屏幕共享）（风险等级：高）

![#设置六：检查网站权限（摄像头、麦克风、](/images/tips/chrome-privacy-settings-guide/body1.jpg)

**不改会怎样：** 如果你曾经不小心允许某个网站访问摄像头或麦克风，该网站可以在你不知情的情况下偷偷开启设备。

**操作：** chrome://settings/privacy → "网站权限" → 逐项检查：
- **摄像头**：移除所有不需要的授权
- **麦克风**：移除所有不需要的授权
- **屏幕共享**：移除所有不需要的授权

**建议：** 保持这三个权限的"不允许网站查看"默认状态。只有在明确需要（如视频会议）时，再临时授权给特定网站。

## 设置七：管理自动填充和密码（风险等级：中）

**不改会怎样：** Chrome 默认保存你在网站上输入的所有密码和表单数据（地址、电话、邮箱等）。这些数据存在你电脑本地，如果电脑被入侵，数据会泄露。

**操作：**

1. chrome://settings/passwords → 检查"询问是否保存密码"是否开启
2. chrome://settings/autofill → 检查"保存并自动填充付款方式"和"保存并自动填充地址"
3. 如果你不希望 Chrome 保存密码，关闭"询问是否保存密码"，改用专门的密码管理器（如 Bitwarden、1Password）

自动填充的安全风险和最佳配置参考 [Chrome 自动填充功能详解](/tips/chrome-autofill/)。

**Chrome 内置密码管理器 vs 专业密码管理器：** Chrome 内置的密码管理器功能够用但安全性一般——密码以明文存储在本地数据库中（虽然系统有加密，但不如专业工具的安全级别）。如果你有大量重要账号（银行、邮箱、社交平台），建议使用 Bitwarden 或 1Password 等专业工具。

## 设置八：关闭"使用预测性服务来帮助搜索"（风险等级：低）

**不改会怎样：** Chrome 会根据你的浏览历史、搜索记录和书签来预测你要搜索的内容，在地址栏输入时显示建议。方便但会进一步暴露你的兴趣偏好。

**操作：** chrome://settings/privacy → "其他" → 关闭"使用预测性服务来帮助搜索"

**效果：** 关闭后地址栏的搜索建议会变少（只显示历史记录匹配），但你的搜索偏好不再被 Google 收集。

## 设置九：了解无痕模式的真正能力（风险等级：中）

很多人对无痕模式有误解，以为开启后就是"完全匿名浏览"。这不对。

**无痕模式实际做的事：**
- 不在本地保存浏览记录
- 不保存 Cookie（关闭窗口后自动删除）
- 不保存表单数据

**无痕模式不做的事：**
- 不能隐藏你的 IP 地址（你的网络服务商仍然能看到你访问了什么网站）
- 不能防止网站追踪你的指纹信息（浏览器版本、屏幕分辨率等）
- 不能隐藏你的 DNS 请求（你的 DNS 服务商知道你查询了什么域名）
- 公司/学校网络管理员仍然可以看到你的网络流量

无痕模式的完整能做什么不能做什么，参考 [Chrome 无痕模式详解](/tips/chrome-incognito-mode-guide/)。

**操作：** Ctrl + Shift + N 打开无痕窗口

**什么时候该用无痕模式：**
- 在公共电脑上登录账号
- 搜索敏感信息（医疗、法律、金融）
- 登录你不信任的网站
- 测试某个网站但不希望被记录

**什么时候不必用：**
- 日常浏览（无痕模式不影响隐私保护，只是不留本地记录）

## 总结：优先级排序

| 优先级 | 设置 | 风险 | 2分钟搞定？ |
|--------|------|------|-----------|
| 🔴 必须改 | 关闭第三方Cookie | 高 | ✅ |
| 🔴 必须改 | 位置信息设为不允许 | 高 | ✅ |
| 🔴 必须改 | 检查摄像头/麦克风权限 | 高 | ✅ |
| 🔴 确认 | 安全浏览已开启 | 高 | ✅ |
| 🟡 建议改 | 清理通知权限 | 中 | ⚠️ 需要5分钟 |
| 🟡 建议改 | 清理浏览数据 | 中 | ✅ |
| 🟡 建议改 | 管理密码保存 | 中 | ⚠️ 需要思考 |
| 🟢 可选 | 关闭预测性搜索 | 低 | ✅ |
| 🟢 了解 | 无痕模式能力 | 低 | ✅ |

前 3 个设置建议你现在就去改，2 分钟就能搞定。后面的设置可以根据你的隐私需求逐步调整。

如果你对隐私保护有更高要求（比如防止浏览器指纹追踪），建议安装 uBlock Origin（广告拦截 + 反追踪）和 Privacy Badger（自动学习并阻止追踪器）这两个扩展。如果你在考虑是否换成其他浏览器，可以参考 [Chrome vs Firefox 2026 深度对比](/compare/chrome-vs-firefox-2026/) 中的隐私保护对比部分。

## 常见问题

### 改了这些设置之后 Chrome 会变慢吗？

不会。关闭第三方 Cookie 和安全浏览不会影响页面加载速度。清理浏览数据后第一次加载某些网站可能稍慢（需要重新加载资源），之后就会恢复正常。

### Chrome 和 Firefox 谁的隐私保护更好？

Firefox 默认的隐私保护比 Chrome 强很多——它默认关闭第三方 Cookie、启用严格追踪保护、开启加密 DNS。如果你非常在意隐私但不想手动改设置，Firefox 是更省心的选择。详见我们的浏览器对比文章。

### 无痕模式能不能防止网监？

不能。如果你使用公司网络、学校网络或公共 Wi-Fi，网络管理员可以通过网络设备看到你的所有流量。无痕模式只保护本地数据不被保存，不保护网络层面的监控。如果需要网络层面的隐私保护，需要使用 VPN。

### Chrome 会把我的浏览数据卖给广告商吗？

Chrome 不会"卖"数据，但 Google 会使用你的浏览数据来投放精准广告（这是 Google 的核心商业模式）。关闭第三方 Cookie 和预测性搜索可以减少 Google 收集到的数据量，但不能完全阻止——只要你还登录了 Google 账号，Google 就会收集你的搜索记录和使用数据。如果这让你不舒服，考虑使用不登录 Google 账号的浏览器（如 Firefox）。

### 为什么不推荐安装隐私保护插件？

隐私保护插件本身也可能收集你的数据。安装太多隐私插件反而增加了被追踪的风险。建议只用 1-2 个信誉良好的插件（uBlock Origin 和 Privacy Badger），不要装 5-6 个同类插件。
