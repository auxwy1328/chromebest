"Chrome 怎么下载视频"是最常被问到的问题之一。答案不是简单的"装个插件就行"——不同网站的视频下载方式往往不同，有的用插件就能搞定，有的需要更复杂的方案。

这篇文章详细整理了 5 种 Chrome 下载视频的方法，从最简单到最专业，覆盖各种不同的使用场景和需求。

## 方法一：Chrome 插件（最简单）

适合下载公开网站（B站、YouTube、Twitter）上的普通视频。

**推荐插件：**

**Video DownloadHelper**

支持最多的视频网站之一。安装后，访问有视频的页面时，插件图标会亮起，点击即可选择下载视频（可选不同分辨率和格式）。支持 HLS 和 DASH 流媒体视频下载。

- 优点：支持的网站多（YouTube、Vimeo、Dailymotion、Facebook 等）、操作简单（检测到视频后一键下载）、免费版功能够用
- 缺点：部分网站的加密视频（如 Netflix、Disney+）无法下载、免费版有每日下载次数限制（约 10 次）

**猫抓（猫抓网页媒体嗅探）**

国内开发者的视频嗅探工具，专门针对国内视频网站优化。安装后自动检测页面中的视频和音频资源，显示在弹窗中，点击即可下载。

- 优点：对国内视频网站（B站、优酷、爱奇艺、腾讯视频、抖音）的支持非常好、完全免费没有任何限制、更新频率高（作者活跃维护）
- 缺点：对国外网站（YouTube、Twitter）的支持不如 Video DownloadHelper、界面比较简陋（功能够用但不太好看）

**注意事项：** Chrome 官方商店已经下架了大部分视频下载插件（因为违反了 Google 的政策——不允许下载受版权保护的内容）。你可以在 Chrome Web Store 搜索"video downloader"，找到的插件可能功能有限。更完整的视频下载插件通常需要从第三方来源安装（如各种 Chrome 插件下载站）。其他必备的 Chrome 插件推荐参考 [Chrome 必装插件推荐 2026](/plugins/chrome-essential-extensions/)。

## 方法二：在线下载网站（不需要安装任何东西）

![#方法二：在线下载网站（不需要安装任何东](/images/tips/chrome-video-download-guide/body3.jpg)

如果你只是偶尔下载一两个视频，不想装插件也不想学命令行，在线下载网站是最方便的选择——打开网页、粘贴链接、点击下载，三步搞定。

**常用在线下载网站：**

| 网站 | 支持平台 | 特点 |
|------|---------|------|
| savefrom.net | YouTube、Instagram、Facebook | 支持选择分辨率，速度快 |
| ssyoutube.com | YouTube | 输入 ytb 后加 URL 即可 |
|.snappe.net | 多平台 | 支持批量下载 |
| bilibili.jijidown.com | B站 | 专门针对B站，支持选择清晰度 |
| tools.liumingye.cn | B站、YouTube、微博、抖音 | 国内开发者做的工具，界面简洁，服务器在国内所以速度较快，免费无广告 |

**使用方法：** 复制视频页面的完整 URL（确保复制了包含视频标识符的完整链接）→ 粘贴到在线下载网站的输入框中 → 等待网站解析视频信息 → 选择你需要的分辨率和格式 → 点击下载按钮。

**优点：** 不需要安装任何东西、在公共电脑上也能用。
**缺点：** 部分网站有弹窗广告或页面广告（注意不要误点下载广告伪装的按钮）、视频质量选择有限（通常只提供 720p 和 1080p，4K 不支持）、下载速度取决于在线网站的解析服务器和网络状况、部分网站需要关注微信公众号或付费开通会员才能下载高清画质。

## 方法三：yt-dlp 命令行工具（最强大）

yt-dlp 是目前最强大的开源视频下载命令行工具，基于已经停止维护的 youtube-dl 项目 fork 而来，支持超过 1000 个视频和音频网站。它的开发非常活跃，几乎每天都有更新，能快速适配各网站的防下载机制变更。虽然需要命令行操作，但功能远超任何插件。

**安装方法：**
```bash
pip install yt-dlp
# 或下载 Windows 可执行文件
# https://github.com/yt-dlp/yt-dlp/releases
```

**常用命令：**
```bash
# 下载视频（自动选择最佳画质）
yt-dlp "视频URL"

# 下载指定分辨率
yt-dlp -f "bestvideo[height<=720]+bestaudio" "视频URL"

# 下载整个播放列表
yt-dlp -o "%(playlist_index)s-%(title)s.%(ext)s" "播放列表URL"

# 只下载音频（提取MP3）
yt-dlp -x --audio-format mp3 "视频URL"

# 指定下载目录
yt-dlp -o "D:\Downloads\%(title)s.%(ext)s" "视频URL"
```

**优点：** 支持网站最多（1000+）、下载质量最高（支持4K）、可以下载字幕、支持播放列表批量下载、完全免费开源、持续更新。
**缺点：** 需要命令行操作（对非技术用户不友好）、没有图形界面。

**如果你想要图形界面：** 可以用 **uydl**（yt-dlp 的图形前端）或 **ClipGrab**（自带界面的视频下载工具，基于 yt-dlp）。

## 方法四：浏览器用户脚本（Tampermonkey）

![#方法四：浏览器用户脚本（Tamperm](/images/tips/chrome-video-download-guide/body2.jpg)

通过 Tampermonkey 安装特定网站的视频下载脚本，可以绕过网站的前端限制直接获取视频的真实下载地址。这种方式的原理是脚本会分析页面的网络请求，找到视频文件的真实 URL（通常是 m3u8 或 mp4 链接），然后在页面上添加一个下载按钮。

**常用脚本（在 GreasyFork 搜索安装）：**

- **Bilibili Evolved**：B站增强脚本，支持视频下载、弹幕下载、播放器增强等功能
- **YouTube Auto HD + FPS**：YouTube 增强，可配合其他下载工具使用
- **Video Download Script**：通用视频下载脚本，自动在视频页面添加下载按钮

**安装方法：** 先安装 Tampermonkey 插件 → 打开脚本页面 → 点击"安装" → 刷新视频页面 → 页面上会出现下载按钮。如果扩展安装遇到问题，参考 [Chrome 扩展程序故障排查指南](/tips/chrome-extensions-not-working/)。

**优点：** 和浏览器集成度高（直接在页面上操作）、支持特定网站的独有功能（如B站弹幕下载）。
**缺点：** 脚本可能因网站改版而失效（需要等作者更新）、部分脚本质量参差不齐、安装第三方脚本有安全风险。

## 方法五：第三方下载软件（IDM、迅雷）

如果你经常下载大文件（不仅是视频，还包括软件安装包、文档、图片等），安装一个专门的下载管理器可能是最全面的方案。下载管理器不只能加速下载速度，还能管理下载队列、自动分类文件、支持断点续传。

**Internet Download Manager (IDM)**

IDM（Internet Download Manager）是 Windows 上最强大的下载管理器，安装后会自动集成到 Chrome 和其他主流浏览器中。它通过接管浏览器的下载请求来实现加速和增强功能。当页面中有可下载的视频时，IDM 会在视频上方显示"下载此视频"的浮动按钮。

- 优点：下载速度快（多线程）、自动接管浏览器下载、支持队列下载、支持断点续传
- 缺点：付费软件（约 25 美元/年）、体积较大

如果 Chrome 本身下载速度就慢（不只是视频），参考 [Chrome 下载速度慢修复指南](/tips/chrome-slow-download-fix/)。

**迅雷**

国内最流行的下载工具，类似 IDM 但更本土化。支持浏览器集成、下载视频、BT/磁力下载等。

- 优点：国内速度快、支持资源丰富、支持浏览器集成
- 缺点：广告多、后台进程多、免费版有速度限制

## 常见问题

### 下载加密视频（VIP 视频）合法吗？

从法律角度，下载受版权保护的视频供个人观看（离线观看）在很多国家的版权法中属于灰色地带。但下载 VIP 视频然后传播或商业使用是明确违法的。本文介绍的方法仅供学习和个人使用，请尊重内容创作者的版权。

### 为什么有的视频无法下载？

三种常见原因：视频使用了加密流媒体技术（HLS AES-128 加密或 DRM 保护），插件和工具无法解密；视频是实时直播流，不是点播文件；网站检测并阻止了视频下载行为。对于加密视频，合法的离线观看方式是使用平台自带的下载功能（如 YouTube Premium 的离线下载、B站 App 的缓存功能）。

### yt-dlp 下载速度慢怎么办？

yt-dlp 本身不限制速度，速度慢通常是视频服务器限速或网络问题。解决方法：使用代理（某些视频服务器对特定地区限速）、使用 aria2 作为外部下载器（yt-dlp 支持 `--downloader aria2c` 参数，实现多线程下载）、在低峰时段下载。

### 下载的视频格式不兼容怎么办？

下载的视频可能是 MKV、WEBM 等格式，某些播放器不支持。使用 FFmpeg 转换格式：
```bash
ffmpeg -i input.mkv -c:v copy -c:a aac output.mp4
```
FFmpeg 是免费开源的视频处理工具，支持几乎所有视频格式的互相转换，功能非常强大。如果觉得命令行操作太麻烦，可以用格式工厂（Windows 上老牌的图形界面格式转换工具，免费使用）或者 HandBrake（开源视频转码工具，专门用于压缩和转换视频文件，输出质量很高）。

### 下载 B站 视频最推荐哪种方式？

对于 B站视频，推荐使用猫抓插件或 bilibili.jijidown.com 在线工具。猫抓插件的优点是可以直接在 B站 页面上操作，不需要切换到其他网站；在线工具的优点是不需要安装任何东西。如果你需要下载 B站的弹幕或字幕，推荐安装 Tampermonkey 中的 Bilibili Evolved 脚本——它是 B站最强大的增强脚本，除了下载视频外还支持弹幕导出、播放器增强、屏蔽广告等功能。注意 B站的 1080P 以上画质需要登录 B站 账号才能观看和下载。
