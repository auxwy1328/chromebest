"我用了无痕模式，没人能追踪到我。"——这是关于浏览器隐私最常见的误解。

Chrome 无痕模式（也叫隐身模式）的名字容易让人产生误导。它确实有一定的隐私保护作用，但远没有大多数人想象的那么强。这篇文章用实测数据告诉你：无痕模式到底能做什么、不能做什么。

## 无痕模式能做的事

### 1. 不在本地保存浏览记录

这是无痕模式最基本的功能。开启无痕模式后，你在该窗口中访问的所有网站都不会出现在浏览历史中。关闭无痕窗口后，所有浏览记录自动清除。

**实测：** 打开 chrome://history/，确认无痕窗口中访问的网站不会出现在历史记录中。✅ 确认有效。

### 2. 不保存 Cookie

无痕窗口中的 Cookie 会在关闭窗口后自动删除。这意味着你登录的账号、购物车的商品、网站的个人化设置都不会被保存。

**实测：** 在无痕窗口中登录知乎，关闭窗口后重新打开无痕窗口，知乎需要重新登录。✅ 确认有效。

### 3. 不保存表单数据和自动填充

你在无痕窗口中填写的表单（姓名、地址、邮箱等）不会被 Chrome 保存。下次填写同一表单时不会出现自动填充建议。

### 4. 不保存文件下载记录

通过无痕窗口下载的文件不会被记录在 Chrome 的下载历史中。但文件本身仍然保存在你的硬盘上——无痕模式只清除"下载记录"，不会删除已下载的文件。

**实测：** 在无痕窗口下载一个文件后，打开 chrome://downloads/，确认没有下载记录。但去下载目录检查，文件确实存在。✅ 确认有效。

### 5. 独立的扩展环境（默认情况）

Chrome 131 的无痕模式默认不允许扩展运行（你可以手动在 chrome://extensions/ 中为每个扩展开启"在无痕模式下启用"）。这意味着追踪型的扩展在无痕模式下不会生效。

## 无痕模式不能做的事（重点）

![#无痕模式不能做的事（重点）](/images/tips/chrome-incognito-mode-guide/body2.jpg)

### 1. 不能隐藏你的 IP 地址

**实测：** 在无痕模式和普通模式下分别访问 https://whatismyipaddress.com/，显示的 IP 地址完全相同。

无痕模式只影响浏览器的本地存储，不影响网络层面的信息。你的网络服务商（ISP）和网站服务器都能看到你的真实 IP 地址。即使你不登录任何网站、不用任何 Cookie，网站仍然可以通过 IP 地址大致定位你的地理位置（精确到城市级别）和使用的网络服务商。

更值得注意的是，你的 IP 地址本身就是一种"身份标识"。如果你的 ISP 给你分配的是固定 IP，那不管你开不开无痕模式，所有网站都知道是"同一个用户"在访问。

### 2. 不能防止 DNS 查询泄露

每次你访问一个网站，浏览器都会向 DNS 服务器查询该域名的 IP 地址。这个查询请求是无痕模式无法隐藏的。你的 DNS 服务商（通常是你的宽带运营商）知道你查询了哪些域名。

举例：你在无痕模式中访问 `taobao.com`，虽然浏览记录不会被保存，但你的 ISP 的 DNS 服务器会记录一次 `taobao.com` 的查询请求。如果有人（如执法部门）向你的 ISP 调取 DNS 查询日志，你的无痕浏览记录就暴露了。

解决办法：使用加密 DNS（如 Chrome 内置的安全 DNS 功能）或通过 VPN 隧道进行 DNS 查询。

### 3. 不能防止网站指纹追踪

即使你不用 Cookie，网站仍然可以通过以下方式识别你：
- **浏览器指纹**：浏览器版本、操作系统、屏幕分辨率、已安装字体、时区、语言等信息的组合可以唯一标识你。根据 EFF 的研究，超过 83% 的浏览器指纹是唯一的
- **Canvas 指纹**：网站通过 HTML5 Canvas API 让你的浏览器绘制一张图片，不同设备的绘制结果有微小差异，可以用来识别设备
- **WebGL 指纹**：通过 GPU 渲染信息识别设备
- **AudioContext 指纹**：通过音频处理 API 的细微差异识别设备

**实测：** 在无痕模式和普通模式下分别访问 https://coveryourtracks.eff.org/（EFF 的指纹检测工具），两次的指纹结果完全相同。无痕模式对此毫无作用。

这意味着即使你每次都用无痕模式、每次都清除 Cookie，网站仍然能通过指纹识别出"还是你"。

### 4. 不能防止网络管理员监控

如果你在公司、学校或公共 Wi-Fi 环境中使用无痕模式，网络管理员仍然可以通过网络设备（防火墙、路由器、深度包检测设备）看到你的所有网络流量——包括你访问了哪些网站、搜索了什么关键词、发送了什么消息。无痕模式只保护你的电脑上的数据，不保护网络传输中的数据。

如果公司网络使用 HTTPS 解密（即 SSL 中间人），管理员甚至能看到你访问的每个页面的具体内容。这种情况下，唯一有效的保护方案是使用公司网络之外的 VPN。

### 5. 不能防止 ISP 记录你的浏览历史

在很多国家和地区，法律要求网络服务商保留用户的浏览记录一定时间（通常 6-12 个月）。无痕模式无法阻止你的 ISP 记录你访问了哪些网站的域名、访问时间、流量大小等信息。这些记录和浏览器无关，是在网络基础设施层面采集的。

### 6. 不能隐藏你的搜索记录

在无痕模式中使用 Google 搜索，Google 仍然知道你搜索了什么（因为你发送搜索请求时可能已经登录了 Google 账号，或者 Google 通过 Cookie 识别了你——无痕模式中 Cookie 是临时的，但在同一无痕会话中仍然有效）。关闭无痕窗口后虽然本地没有记录，但 Google 服务器上仍然有记录。

## 什么时候该用无痕模式

| 场景 | 无痕模式有用吗 | 说明 |
|------|-------------|------|
| 在公共电脑上登录账号 | ✅ 有用 | 关闭窗口后不会留下登录状态和浏览记录 |
| 搜索敏感信息（医疗、法律） | ⚠️ 部分有用 | 本地不留痕迹，但 ISP 和网站仍然知道 |
| 登录多个账号（如两个 Gmail） | ✅ 有用 | 普通窗口一个账号，无痕窗口另一个账号 |
| 测试网页效果 | ✅ 有用 | 不受已登录状态和缓存影响 |
| 防止被追踪 | ❌ 没用 | IP 和指纹仍然暴露 |
| 防止网监 | ❌ 没用 | 网络层面的监控不受浏览器模式影响 |
| 购物比价（防大数据杀熟） | ⚠️ 部分有用 | 如果登录了账号，商家仍然能识别你 |

## 真正的隐私保护方案

![#真正的隐私保护方案](/images/tips/chrome-incognito-mode-guide/body1.jpg)

如果你需要比无痕模式更强的隐私保护，需要以下组合：

### 基础方案（免费）
- **VPN**：隐藏你的真实 IP 地址和 DNS 查询
- **uBlock Origin**：拦截广告追踪器
- **Privacy Badger**：自动阻止跨站追踪器
- **Firefox + 严格追踪保护**：Firefox 默认的隐私保护比 Chrome 强

### 进阶方案
- **Tor Browser**：通过多层加密的洋葱网络路由流量，提供匿名浏览。适合需要高隐私保护的场景（但不适合日常使用——速度很慢）
- **Mullvad Browser**：基于 Firefox 的隐私浏览器，配合 Mullvad VPN 使用

### 企业级方案
- **代理 + VPS**：自建代理服务器，完全控制网络流量
- **虚拟机 + Tails OS**：在虚拟机中运行 Tails 系统，关机后所有数据自动清除

## 常见问题

### 无痕模式下下载的文件去哪了？

文件保存在你的系统默认下载目录中（通常是 `C:\Users\<用户名>\Downloads\`）。无痕模式不删除已下载的文件，只清除下载记录。如果你想彻底清除文件，需要手动删除。

### 无痕模式会影响网速吗？

不会。无痕模式只是改变了浏览器的存储行为，不影响网络传输。不过由于无痕模式不使用缓存，每次访问网站都需要重新加载所有资源（图片、CSS、JS），所以**页面加载速度可能比普通模式稍慢**。

### 怎么设置无痕模式默认开启某些扩展？

chrome://extensions/ → 找到你想在无痕模式下使用的扩展 → 点击"详情" → 开启"在无痕模式下启用"。只有你手动开启的扩展才会在无痕模式下运行，其他扩展默认被禁用。

### 无痕模式和 VPN 一起用效果怎样？

比单独用无痕模式好很多。VPN 隐藏了你的 IP 地址和 DNS 查询，无痕模式清除了本地数据。两者配合可以覆盖大部分隐私需求。但仍然不能防止浏览器指纹追踪——需要额外安装指纹防护插件（如 Canvas Blocker）。

### Chrome 的无痕模式和其他浏览器的隐私模式有区别吗？

功能上基本相同——都是不保存本地浏览数据、不保存 Cookie、不保存表单数据。但 Firefox 的隐私浏览模式额外集成了追踪保护（默认拦截第三方追踪器），隐私保护比 Chrome 无痕模式更强一些。Brave 的隐私窗口则更进一步——默认拦截广告和追踪器。
