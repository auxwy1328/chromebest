打开一个 PDF 文件时，大多数人会习惯性地下载然后用 Adobe Acrobat 或其他 PDF 软件打开。但 [Chrome 浏览器](/) 本身就内置了一个功能相当完善的 PDF 查看器和编辑器，完全可以胜任日常的 PDF 阅读与编辑需求。本文会详细讲解 [Chrome PDF 编辑功能](/) 的实际能力，包括标注高亮、填写表单、打印保存、以及它的局限性和替代方案。

## Chrome PDF 查看器的基本功能

Chrome 从很早的版本开始就集成了 PDF 查看器，基于 PDFium 渲染引擎（Chromium 项目的开源 PDF 库）。当你点击一个 PDF 链接或打开本地 PDF 文件时，Chrome 不会调用外部程序，而是直接在标签页内渲染 PDF 内容。这一点在很多浏览器中并不常见——Firefox 直到 2024 年才完全内置了原生 PDF 渲染。

Chrome 的 PDF 查看器提供了以下基础功能：

- **缩放**：通过工具栏的加减号按钮、快捷键 Ctrl+和 Ctrl--、或鼠标滚轮调整页面大小
- **页面导航**：上下翻页、跳转到指定页码、缩略图面板快速定位
- **全屏阅读**：按 F11 进入全屏模式，获得沉浸式阅读体验
- **文本选择与复制**：可以直接选中 PDF 中的文字进行复制
- **搜索**：Ctrl+F 在 PDF 全文中搜索关键词，高亮显示匹配结果
- **旋转页面**：对于方向错误的 PDF 页面可以旋转调整
- **下载与打印**：直接下载原始文件或通过打印功能保存

这些基础功能在 [Chrome 浏览器](/) PDF 查看器的顶部工具栏中都可以直接访问。如果想要更全面地了解 Chrome 的操作技巧，可以参考 [Chrome 键盘快捷键大全](/tips/chrome-clear-cache/) 来提升操作效率。

## Chrome PDF 标注功能详解

\[Chrome PDF 标注工具界面](/images/tips/chrome-pdf-edit/annotate.jpg)

Chrome PDF 查看器在近年的更新中加入了标注功能，这是很多用户完全没有注意到却非常实用的特性。标注工具栏默认隐藏，需要手动开启。

### 开启标注工具栏

在 Chrome 中打开 PDF 文件后，有两种方式打开标注面板：

1. **点击工具栏图标**：在 PDF 查看器右上角找到画笔/铅笔形状的图标，点击即可展开标注工具面板
2. **快捷键方式**：按 Shift+Ctrl+A（Windows）或 Shift+Cmd+A（Mac）快速打开

标注面板打开后会在右侧弹出一个垂直工具条，包含多种标注工具。

### 可用的标注工具

Chrome PDF 标注支持以下几种操作：

**高亮标注**：选中 PDF 中的文字后，可以添加黄色、绿色、蓝色、粉色等不同颜色的高亮。适合标记重要段落或需要二次审阅的内容。实际使用中发现，高亮颜色会自动保存，关闭标签页后重新打开依然保留。

**自由文字添加**：选择文字工具后，点击 PDF 页面的任意位置即可在该处输入自定义文字。可以调整字体大小和颜色，适合在空白处添加批注、注释或说明。对于 [Chrome 阅读模式](/tips/chrome-reading-mode/) 用户来说，这个功能弥补了网页转 PDF 后无法编辑的遗憾。

**下划线与删除线**：对选中文字添加下划线或删除线标记，适合审校文档时标出需要修改或删除的内容。

**手绘涂鸦**：用鼠标或触摸板在 PDF 上自由绘制线条和图形。适合在图表上圈注关键数据，或用箭头指向某个特定区域。线条粗细和颜色都可以调整。

**橡皮擦**：撤销已添加的标注，支持逐个删除或一键清除全部标注。

### 标注的保存与导出

添加的标注会自动保存在 Chrome 的本地存储中。但需要注意，这些标注并没有真正写入 PDF 文件本身，而是存储在浏览器的 IndexedDB 数据库中。这意味着：

- 换一台电脑或清除浏览器数据后标注会丢失
- 直接下载（Ctrl+S）的 PDF 文件不包含标注
- 要把标注固化到 PDF 中，必须通过打印为 PDF（Ctrl+P，选择另存为 PDF）来实现

这是一个很多人踩过的坑：辛苦标注了一整天的合同，直接下载后发现标注全没了。记住，打印为 PDF 才是保存标注的正确方式。

## PDF 表单填写

\[在 Chrome 中填写 PDF 表单](/images/tips/chrome-pdf-edit/form-fill.jpg)

Chrome PDF 查看器支持填写包含交互字段的 PDF 表单。这在日常办公中非常实用，例如填写税务表格、申请表、合同模板等。

### 识别可交互表单

当你在 Chrome 中打开一个 PDF 表单时，如果该 PDF 包含可交互的表单字段，你会看到光标悬停在空白处会变为输入指针，点击后出现闪烁的输入光标或可点击的复选框。常见的交互字段类型包括：

- **文本输入框**：用于填写姓名、地址、金额等信息
- **复选框**：勾选或取消勾选选项
- **单选按钮**：在一组互斥选项中选择一个
- **下拉菜单**：从预定义列表中选择一个选项
- **日期选择器**：选择特定日期
- **数字输入框**：只能输入数字的字段

### 填写技巧

填写 PDF 表单时，有几个实用技巧可以提升效率：

Tab 键可以在表单字段之间快速切换，不用每次都用鼠标点击下一个输入框。对于大量文字输入的表格，键盘导航比鼠标操作快得多。

如果表单字段有预设的格式要求（如电话号码、邮政编码），Chrome 会自动进行基本的格式验证。输入不符合格式的内容时，字段可能会有红色边框提示。

填写完成后，强烈建议先预览一遍所有字段，确认没有遗漏。然后通过 Ctrl+P 打印为 PDF 保存填写结果。不要直接关闭标签页，否则填写内容可能会丢失。

### 扫描件表单的处理

需要注意，不是所有 PDF 表单都包含交互字段。很多扫描版 PDF（如扫描的纸质表格）本质上就是一张图片，无法直接点击填写。这种情况下有以下替代方案：

1. **Chrome 标注工具手动填写**：使用文字工具在扫描表格的空白处手动添加文字。效果不如原生表单自然，但可以应急使用
2. **OCR 识别后填写**：使用 Google Drive 上传扫描 PDF，Drive 会自动进行 OCR 识别并转换为可编辑的 Google Docs 文档
3. **专业 PDF 工具**：使用 PDF24 或 Adobe Acrobat 的 OCR 功能，将扫描件转换为可填写表单

关于 Chrome 的更多实用功能设置，可以查看 [Chrome 外观自定义教程](/plugins/chrome-essential-extensions/)。

## 网页打印为 PDF

\[Chrome 打印对话框中选择另存为 PDF](/images/tips/chrome-pdf-edit/print-pdf.jpg)

将网页保存为 PDF 是 Chrome 最常用的 PDF 相关功能之一。无论是保存文章归档、制作报告截图、还是将在线表格导出为文件，打印为 PDF 都是最便捷的方式。

### 操作步骤

在任意网页上按 Ctrl+P（Mac 上为 Cmd+P）打开 Chrome 的打印对话框。在目标打印机选项中选择另存为 PDF，然后点击保存即可。但在保存之前，有几个关键设置值得调整：

**布局选项**：纵向或横向。默认纵向，但宽表格或横向排版的内容需要切换为横向。

**页边距**：默认、无、最小或自定义。选择无可以让内容铺满整页，适合截图类内容；默认边距更适合正式文档打印。

**缩放比例**：默认 100%，可以调整让内容适配纸张大小。如果网页内容过宽导致被裁切，可以缩小到 80% 或 90%。

**背景图形**：默认关闭，勾选后会保留网页的背景颜色和图片。对于需要完整保留网页外观的场景（如制作截图报告），务必勾选此选项。

**页眉和页脚**：可以添加日期、URL、页码等信息到 PDF 的顶部和底部。如果只需要纯净内容，取消勾选即可。

### 高级打印技巧

Chrome 的打印为 PDF 功能还有一些不太为人知的技巧：

**自定义纸张大小**：点击更多设置展开完整选项，可以在纸张大小中选择 A3、A4、Letter、Legal 等多种尺寸，也可以输入自定义尺寸。

**CSS 打印媒体查询**：很多网站会针对打印输出优化样式。Chrome 打印时自动应用 @media print 样式，隐藏不必要的导航栏和广告，只保留正文内容。

**仅打印选中内容**：在网页上先用鼠标选中需要保存的部分文字和图片，然后在打印对话框中勾选仅限选定内容，这样 PDF 只包含选中区域的内容，而不是整个页面。

**PDF 合并**：虽然 Chrome 本身不支持直接合并 PDF，但有一个取巧的方法：在打印对话框的页面范围中选择多页，或者打开多个标签页后逐一打印为 PDF，然后用外部工具合并。更好的方案是使用 PDF24 的合并功能，直接拖拽多个 PDF 文件即可合并为一个。

如果经常遇到 Chrome 页面加载异常导致打印失败的情况，可以参考 [Chrome 页面无法加载的解决方法](/tips/chrome-page-not-loading/) 进行排查。

## Chrome PDF 功能的局限性

Chrome 的 PDF 功能虽然日常够用，但和专业的 PDF 编辑软件相比有明显的功能缺失。了解这些局限可以帮助你在需要时选择更合适的工具。

### 无法修改原始内容

这是最大的限制。Chrome PDF 查看器无法修改 PDF 文件中的现有文字、图片、表格等内容。你不能删除一段话、替换一张图片或修改表格数据。Chrome 只能在 PDF 上层叠添加标注，无法触及底层的 PDF 内容。

对于需要修改 PDF 原文内容的需求，推荐使用 PDF24（免费在线工具）或 LibreOffice Draw（开源桌面软件），两者都支持直接编辑 PDF 的文字和图片。

### 无法删除或重排页面

Chrome 不能删除 PDF 中的某一页，也不能调整页面顺序、插入新页面或将多个 PDF 合并为一个文件。如果你的 PDF 文档有多余的页面需要删除，必须使用外部工具。PDF24 提供了免费的在线页面管理功能，支持删除、旋转、提取和重排 PDF 页面。

### 无法添加或编辑表单字段

Chrome 只能填写已有的交互表单，无法在 PDF 中创建新的表单字段。如果你需要在 PDF 中添加填空题、复选框或下拉菜单，需要使用 Adobe Acrobat 或 PDF24 Pro 等专业工具。

### 标注功能的限制

虽然 Chrome 支持添加标注，但相比专业 PDF 标注工具有以下不足：

- 不支持添加附件（如将其他文件附到 PDF 某一页上）
- 不支持添加矩形、圆形、线条等标准图形
- 不支持撤销单个标注的历史记录（只能用橡皮擦逐个删除）
- 不支持标注的导出导入（换电脑标注就没了）
- 手绘线条不够平滑，不适合精细的审校工作

### 文件大小限制

对于特别大的 PDF 文件（超过 100MB），Chrome 的 PDF 查看器可能会变得卡顿甚至无法打开。这是因为 PDFium 引擎需要将整个文件加载到内存中。遇到这种情况，可以尝试用 [Chrome 内存优化技巧](/tips/chrome-hardware-acceleration/) 释放系统资源，或者改用桌面端 PDF 阅读器。

## Chrome 扩展增强 PDF 功能

\[不同 PDF 工具的对比](/images/tips/chrome-pdf-edit/tools-comparison.jpg)

如果你需要更强大的 PDF 编辑能力但又不想离开浏览器，Chrome 扩展是很好的补充。以下是一些经过实际测试的优质扩展：

### PDF Extra - PDF 编辑器

PDF Extra 是一个功能丰富的 Chrome 扩展，在浏览器标签页内直接提供接近桌面级 PDF 编辑能力。支持修改 PDF 文字、添加图片、插入页面、填写表单、添加电子签名等操作。免费版支持基本的阅读和标注功能，高级编辑功能需要订阅。

优点是界面和 Google Drive PDF 查看器类似，学习成本低。缺点是免费版有水印，且文件大小限制在 10MB 以内。

### Kami - PDF 标注与协作

Kami 是一款以协作标注为核心的 PDF 扩展。除了标准的高亮、下划线、文字标注外，它还支持实时协作——多人同时打开同一个 PDF 文件进行标注和批注，适合团队审阅合同或学术论文。

Kami 提供语音批注、视频批注、屏幕截图标注等特色功能。教育用户可以免费使用，商业用户需要订阅。与 Google Classroom 和 LMS 平台的深度集成也是它的一大优势。

### DocuSign - 电子签名

如果你需要经常签署 PDF 文档，DocuSign 是最成熟的电子签名扩展。它提供法律认可的电子签名，支持拖放签名、自动填充签名信息、发送签署请求等功能。

免费账户每月有 5 次签署额度，对于偶尔签合同的用户来说足够了。DocuSign 的签名在法律上等同于手写签名，被全球 100 多个国家和地区认可。

### Smallpdf

Smallpdf 是一个集成了多种 PDF 工具的扩展，包括压缩、转换、合并、拆分、加水印、加密码等功能。它的优势是操作极简，基本上每个功能都是上传文件后一键完成。

缺点是免费用户每天只能处理 2 个文件，且文件大小限制在 5MB。适合轻度使用，重度用户不如直接安装 PDF24 桌面版。

关于 Chrome 扩展的安装和管理，可以参考 [Chrome 必备扩展推荐](/plugins/chrome-essential-extensions/) 一文。

## 免费 PDF 工具替代方案

当 Chrome 的内置功能不够用时，以下免费工具是最实用的替代选择：

### PDF24 Tools

PDF24 是目前最全面的免费 PDF 工具套件。它提供在线版和桌面版两种使用方式。在线版通过浏览器直接访问，无需注册。桌面版支持 Windows、macOS 和 Linux。

PDF24 的核心功能包括：

- **合并 PDF**：拖拽多个 PDF 文件到页面上，一键合并为一个文件
- **拆分 PDF**：按页码范围或每页单独拆分
- **压缩 PDF**：将大文件压缩到更小体积，适合邮件发送
- **PDF 转 Word/Excel/PPT**：高质量格式转换，保留排版
- **编辑 PDF**：直接修改 PDF 中的文字、图片和链接
- **添加签名**：手写签名或上传签名图片
- **添加水印**：批量给 PDF 添加文字或图片水印
- **PDF 加密**：设置密码保护，限制打印和编辑权限

PDF24 的桌面版功能比在线版更强大，且没有文件大小限制。对于需要频繁处理 PDF 的用户，强烈建议安装桌面版。

### LibreOffice Draw

很多人不知道 LibreOffice 的 Draw 组件可以直接打开和编辑 PDF 文件。它可以修改 PDF 中的文字、替换图片、调整排版、添加图形和标注。虽然是矢量编辑工具，对于简单的内容修改完全胜任，而且完全免费开源。

缺点是对复杂排版的 PDF 支持不够好，特别是包含大量图表和特殊字体的文件可能会出现排版偏移。

### Microsoft Edge 的 PDF 功能

如果你使用的是 Windows 系统，Edge 浏览器的 PDF 查看器其实比 Chrome 更强大。Edge 基于 PDFium 的一个扩展版本，额外支持：朗读 PDF 内容（大声朗读功能）、手写笔输入（支持 Surface 等触控设备直接在 PDF 上书写）、更灵活的标注选项、墨迹书写与擦除。

如果你经常用触控笔在 PDF 上做笔记，Edge 可能比 Chrome 更合适。

## 常见问题排查

### PDF 在 Chrome 中打开是空白

这是最常见的 Chrome PDF 问题之一。原因和解决方法如下：

**硬件加速冲突**：Chrome 的硬件加速功能与某些显卡驱动不兼容，导致 PDF 渲染失败。在 Chrome 设置中搜索硬件加速，关闭该选项后重启浏览器。关闭后 PDF 渲染会使用 CPU 而非 GPU，虽然稍慢但更稳定。关于硬件加速的详细排查可以参考 [Chrome 硬件加速设置指南](/tips/chrome-hardware-acceleration/)。

**PDFium 插件被禁用**：在地址栏输入 chrome://plugins/，确认 Chrome PDF Viewer 插件处于启用状态。如果被禁用，PDF 文件会被直接下载而不是在浏览器中打开。

**缓存问题**：清除 Chrome 缓存和 Cookie 后重新打开 PDF。按 Ctrl+Shift+Delete 打开清除浏览数据窗口，选择缓存的图片和文件后清除。详细的清理方法可以查看 [Chrome 清除缓存教程](/tips/chrome-clear-cache/)。

### PDF 显示乱码或中文缺失

乱码问题通常与字体缺失有关：

- **嵌入字体**：如果 PDF 创建时没有嵌入中文字体，而你的电脑上恰好没有安装对应字体，Chrome 会使用系统默认字体替代，导致乱码
- **解决方案**：安装 Adobe 字体包或常用中文字体（如思源黑体、微软雅黑），然后重新打开 PDF
- **GB2312 编码问题**：一些老旧的中文 PDF 使用 GB2312 编码，Chrome 的 PDFium 引擎对这种编码支持有限。可以尝试下载后用 Adobe Reader 打开

### PDF 被下载而不是预览

正常情况下 Chrome 应该在标签页内预览 PDF，但有时候会直接触发下载。排查方法：

1. 检查 Chrome 设置：进入设置中的隐私与安全，找到网站设置中的 PDF 文档，确保设置为在 Chrome 中打开 PDF 文件
2. 检查 Content-Disposition：某些网站在响应头中设置了 Content-Disposition: attachment，强制浏览器下载而非预览。这需要网站管理员修改服务器配置
3. 文件关联问题：如果系统默认的 PDF 程序不是 Chrome，可能会自动下载。在 Windows 设置的应用关联中检查 .pdf 文件的默认打开方式

### Chrome PDF 查看器崩溃

如果 Chrome 在打开某些 PDF 时频繁崩溃，尝试以下方法：

- 禁用所有扩展后测试，排除扩展冲突
- 更新 Chrome 到最新版本
- 清除 Chrome 的 IndexedDB 数据（这会丢失已保存的标注）
- 在地址栏输入 chrome://settings/pdfDocuments，重置 PDF 文档设置

如果以上方法都不起作用，可能是特定 PDF 文件的编码问题导致 PDFium 解析失败。尝试用其他 PDF 阅读器打开该文件确认是否为文件本身的问题。
