Chrome 手机版的用户量超过桌面版——全球有超过 30 亿人用手机 Chrome 上网。但大部分手机用户只用 Chrome 做最基本的浏览操作，忽略了它很多实用功能。

这篇文章整理了 Chrome 手机版（安卓和 iOS）的 15 个实用技巧，按使用场景分类。每个技巧都标注了安卓和 iOS 的支持情况。

基础操作技巧

1. 快速切换标签页

安卓： 在地址栏下方水平滑动可以切换标签页——向左滑切换到左边一个标签页，向右滑切换到右边一个标签页。这比点击标签页图标再选择目标标签页快得多。

iOS： iOS 版 Chrome 不支持滑动切换标签页，需要点击底部工具栏的标签页图标（方形叠放图标）来切换。

2. 长按标签页预览内容

安卓： 在标签页视图中（点击标签页图标），长按某个标签页可以预览该页面的缩略图。适合标签页开多了不知道哪个是什么的时候快速定位。

iOS： 同样支持，长按标签页卡片即可预览。

3. 一键关闭所有标签页

安卓/iOS： 打开标签页视图 → 点击右上角三个点 → 选择"关闭所有标签页"。Chrome 会提示你是否保留当前标签页。建议开启"关闭所有标签页前询问"功能（设置 → 隐私和安全 → 关闭所有标签页前询问），防止误操作。

4. 下拉刷新

安卓/iOS： 在页面顶部向下拉动即可刷新页面。和原生 App 的操作方式一致，不需要点击菜单找刷新按钮。如果页面有固定的顶部导航栏，需要在导航栏下方的空白区域下拉。

隐私和安全技巧

5. 快速切换无痕模式

安卓： 长按 Chrome 的应用图标（在桌面或应用抽屉中），在弹出的快捷菜单中选择"新建无痕标签页"。或者打开 Chrome 后点击三个点 → “新建无痕标签页”。

iOS： 长按 Chrome 图标 → “新建无痕标签页”。或者在 Chrome 内点击三个点 → “新建无痕标签页”。

小技巧： 如果你经常需要切换普通模式和无痕模式，可以在 Chrome 设置中启用"无痕模式锁定"功能——开启后切换回普通模式需要用指纹或面部识别验证，防止别人在你手机上关闭无痕模式看到历史记录。关于无痕模式能做什么不能做什么，参考 Chrome 无痕模式详解。

6. 指纹锁定 Chrome

安卓： 设置 → 隐私和安全 → 锁定 Chrome → 选择"锁屏时"或"离开 Chrome 时"。开启后，每次打开 Chrome 都需要指纹验证。

iOS： Chrome 130+ 支持 Face ID 锁定。设置 → 隐私和安全 → 锁定 Chrome。

这个功能在手机借给别人使用时特别有用——别人能打开 Chrome 但看不到你之前的标签页和浏览记录。

7. 清除特定网站的缓存

安卓/iOS： 设置 → 隐私和安全 → 清除浏览数据 → 选择"时间范围"为"过去 1 小时"或"过去 24 小时" → 只勾选"缓存的图片和文件"和"Cookie"。这样只清除近期的数据，不会影响其他时间段的登录状态。

效率提升技巧

8. 桌面版网站模式

安卓： 点击三个点 → 勾选"桌面版网站"。页面会以电脑端布局显示（类似在电脑浏览器中查看的效果）。适合浏览只提供电脑版功能的网站（某些后台管理系统、在线编辑器等）。

iOS： 三个点 → “请求桌面网站”。

注意： 桌面版模式会消耗更多流量（因为加载的是桌面版页面，通常比手机版更大），在流量有限的情况下慎用。

9. 离线下载页面

安卓/iOS： 点击三个点 → 下载图标（向下的箭头）→ 选择"当前页面"或"链接"。下载后的页面可以在离线状态下阅读。

适用场景： 地铁上、飞机上等没有网络的环境下阅读之前保存的网页。下载的页面会显示在 Chrome 的"下载"页面中（三个点 → 下载内容）。

10. 预加载网页（加速打开）

安卓： 设置 → 隐私和安全 → 预加载网页 → 选择"仅 Wi-Fi"或"始终"。开启后，当 Chrome 预测你可能要打开某个链接（比如你在搜索结果页面停留时），会提前在后台加载该页面。点击链接时页面会瞬间打开。

iOS： 设置 → 带宽 → 预加载网页。

注意： 开启预加载会消耗更多数据流量（即使你最终没有点击该链接）。建议只在 Wi-Fi 环境下开启。

搜索和导航技巧

11. 地址栏直接搜索

安卓/iOS： 点击地址栏输入框，直接输入关键词即可搜索（不需要先打开搜索引擎网站）。Chrome 默认会把你输入的内容发送给当前设置的搜索引擎。

快速切换搜索引擎： 在地址栏中输入关键词后，长按搜索按钮（放大镜图标），会弹出可选的搜索引擎列表（Google、百度、Bing 等），可以快速切换。

12. 语音搜索

安卓： 点击地址栏右侧的麦克风图标，对着手机说话即可搜索。

iOS： 点击地址栏中的麦克风图标（需要授权 Chrome 使用麦克风）。

适用场景： 走路时、做饭时等不方便打字的场景。Chrome 的语音搜索在中文识别准确率上已经很高，日常搜索基本没问题。

13. 长按图片快速搜索

安卓/iOS： 长按网页中的任何图片，选择"用 Google 搜索图片"。Chrome 会打开一个新标签页，显示该图片的搜索结果——包括相似图片、图片来源、相关信息等。

适用场景： 看到一件好看的衣服想知道在哪里买、看到一张风景照想知道是哪里拍的、看到一张不知名的动植物想知道品种等。

标签页管理技巧

14. 标签页分组

安卓： Chrome 130+ 支持标签页分组。在标签页视图中，长按一个标签页然后拖到另一个标签页上，两个标签页会自动合并为一个分组。可以给分组命名。

iOS： 暂不支持标签页分组功能。

15. 最近关闭的标签页

安卓/iOS： 点击三个点 → “最近关闭的标签页”。Chrome 会显示最近关闭的标签页列表（包括设备间的跨设备关闭记录，如果开启了同步）。点击即可恢复。

安卓专属功能

Chrome 安卓版的额外技巧

底部地址栏： Chrome 安卓版 130+ 支持将地址栏移到底部（设置 → 工具栏位置 → 底部）。对于大屏手机用户来说，底部地址栏更方便单手操作。

自定义启动页： 打开新标签页时，Chrome 会显示常用网站、最近关闭的标签页和文章推荐。你可以在设置 → 主页中自定义新标签页的内容。

静默通知权限： 当网站请求发送通知时，Chrome 安卓版会在地址栏下方显示提示，而不是弹出对话框。如果你不想被通知打扰，可以关闭"网站可以发送通知"权限（设置 → 网站设置 → 通知）。

iOS 专属功能

Chrome iOS 版的额外技巧

Safari 底层渲染： Chrome iOS 版使用的是苹果的 WebKit 引擎（和 Safari 一样），而不是 Chrome 桌面版的 Blink 引擎。这意味着某些只在 Blink 引擎中支持的功能在 Chrome iOS 版上不可用。但这不影响绝大多数网页的正常显示和功能。

与 Safari 共享密码： Chrome iOS 版可以访问 iCloud 钥匙串中保存的密码。设置 → 密码 → 使用已保存的密码 → 启用"从 iCloud 钥匙串填充密码"。

常见问题

Chrome 手机版和桌面版可以同步吗？

可以，但需要登录同一个 Google 账号，并且需要网络工具来访问 Google 的同步服务器（国内环境下同步功能不可用）。可以同步的内容包括：书签、密码、历史记录、已打开的标签页、支付方式和地址信息。注意扩展程序不能在手机版上使用（手机版不支持桌面版插件）。同步的详细设置参考 Chrome 同步设置完整指南。

手机版 Chrome 能装广告拦截插件吗？

安卓版 Chrome 不支持传统的桌面版扩展程序。但你可以通过以下方式实现广告拦截：安装 Chrome 安卓版自带的"广告拦截"实验性功能（chrome://flags → 搜索"Ad Block"），或者使用支持广告拦截的第三方浏览器（如 Brave、Firefox）。iOS 版 Chrome 也不支持扩展程序，但 Safari 支持内容拦截器扩展。

手机版 Chrome 占用存储空间太大怎么办？

Chrome 的存储占用主要来自缓存数据和下载的文件。清理方法：设置 → 隐私和安全 → 清除浏览数据 → 勾选"缓存的图片和文件"→ 清除数据。另外检查"下载内容"页面，删除不需要的离线页面和下载文件。如果 Chrome 仍然占用大量空间，可以在手机设置 → 应用 → Chrome → 存储 → 清除缓存。桌面版 Chrome 的缓存清理方法参考 Chrome 缓存清理教程。

如何在手机上查看网页的 HTML 源代码？

安卓版 Chrome 支持查看源代码：在地址栏中输入 view-source: 加上目标网址（例如 view-source:https://www.example.com），回车后即可查看该页面的 HTML 源代码。iOS 版 Chrome 暂不支持此功能。

手机版 Chrome 有开发者工具吗？

没有。手机版 Chrome 不支持桌面版的完整开发者工具（F12 面板）。但你可以通过 USB 连接电脑，在电脑版 Chrome 的 chrome://inspect 页面中远程调试手机上的页面。操作方法：手机 Chrome → 设置 → 启用"USB 调试" → USB 连接电脑 → 电脑 Chrome 访问 chrome://inspect → 点击"inspect"。
