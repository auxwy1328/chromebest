你可能在用 ChatGPT、Claude、文心一言——但你知道 Chrome 浏览器自己就内置了 AI 吗？

2026年，Google 把 Gemini Nano 模型直接集成进了 Chrome 浏览器。不需要安装任何插件，不需要注册任何账号，不需要联网——选中文本就能总结、翻译、改写、解释。

重点：这是离线 AI。所有处理在你的电脑上完成，数据不离开设备。

58 篇 Chrome 教程里，这个功能一直没讲到。补上。

一、Chrome 内置 AI 是什么

Gemini Nano vs 云端 Gemini 的区别

特性 Chrome 内置 Gemini Nano 云端 Gemini
运行位置 本地设备 Google服务器
联网要求 仅首次下载模型时需要 必须联网
模型大小 约1.5GB 未知（远大于Nano）
隐私 数据不离开设备 数据经过Google服务器
能力水平 轻量任务（总结/翻译/改写） 全能（推理/代码/创作）
费用 免费 部分功能需订阅
响应速度 极快（本地处理无延迟） 取决于网络

简单理解：Gemini Nano 是"浏览器里的迷你AI助手"，不是"浏览器里的ChatGPT"。它做轻量任务很好用，复杂推理还是得用云端大模型。

Chrome AI 的 5 个核心功能

  1. Tab总结：一键总结当前网页内容，生成3-5条要点
  2. 文本改写：选中任意文本，让AI帮你改写得更正式/简洁/友好
  3. 翻译增强：比传统Google翻译更自然的语意翻译
  4. 术语解释：选中不懂的词汇，AI直接在侧边栏给出解释
  5. 写作辅助：在任意文本框中（邮件、评论、表单），让AI帮你起草内容

二、三步开启 Chrome 内置 AI

第 1 步：确认 Chrome 版本

地址栏输入 chrome://settings/help，Chrome 会自动检查更新。

需要版本 138 或以上。如果版本低于138，等待自动更新完成（通常1-2分钟），然后点击"重新启动"。

第 2 步：开启实验功能

地址栏输入 chrome://flags，在搜索框中输入 AI。

你会看到几个相关选项：

  • Optimization Guide On Device Model → 设为 Enabled
  • Chrome AI features → 设为 Enabled（如果存在）
  • Gemini Nano → 设为 Enabled（如果存在）

设置完成后，点击底部的"Relaunch"按钮重启 Chrome。

第 3 步：首次使用时下载模型

重启 Chrome 后，打开任意网页，选中文本，右键菜单中会出现"AI选项"或"用AI总结"。

首次点击时，Chrome 会在后台下载 Gemini Nano 模型文件（约1.5GB）。下载进度可以在 chrome://components 页面查看（搜索"Optimization Guide"）。

下载完成后（通常5-15分钟，取决于网速），AI 功能即可离线使用。

验证是否成功

在任意网页选中文本 → 右键 → 如果看到"用AI总结"、“改写"等选项，说明已成功开启。

如果没看到这些选项：

  • 确认设备内存 ≥ 4GB（推荐8GB以上）
  • 确认硬盘剩余空间 ≥ 3GB
  • 确认 Chrome 版本 ≥ 138
  • 尝试重启 Chrome

三、5 大使用场景实测

场景 1：一键总结长文

打开一篇长网页（新闻、博客、技术文档），选中正文区域，右键 →“AI总结”。

侧边栏会弹出3-5条要点摘要，通常在2秒内生成（本地处理无网络延迟）。

实测效果：

  • 中文新闻文章：准确度约85%，核心信息覆盖完整
  • 英文技术文档：准确度约90%，翻译+总结一次完成
  • 长篇社论：会丢失部分细节，建议分段总结

场景 2：改写选中文本

在任意文本框中（邮件、评论区、文档），输入一段话，选中它，右键 →“用AI改写”。

Chrome 会弹出选项：正式/简洁/友好/长/短。选择风格后，AI 在原位替换文本。

实测效果：

  • 把口语化的"这个方案我觉得不太行"改成正式的"该方案的可行性有待商榷”——效果不错
  • 控制字数（让长文本变短）——效果一般，有时候只删了几个字

场景 3：翻译增强

选中网页上的外语文本，右键 →“AI翻译”。

和传统 Google 翻译的区别：Gemini Nano 会理解上下文后翻译，不是逐句机翻。

实测效果：

  • 英文技术文章翻译成中文：质量接近 DeepL，优于 Google 翻译
  • 中文翻译成英文：语法正确但用词偏简单
  • 日文翻译成中文：质量一般，建议用专业翻译工具

场景 4：术语解释

选中网页上不懂的词汇或概念，右键 →“AI解释”。

侧边栏会弹出一段简短解释，不需要跳转到搜索引擎。

实测效果：

  • 技术术语（如"Service Worker"、“WebAssembly”）：解释准确且简洁
  • 专业概念（如"量化宽松"）：解释基本准确但不够深入
  • 缩写（如"ROI"）：能识别常见缩写并给出展开

场景 5：邮件/评论起草

在 Gmail 或任意网页的文本框中，右键 →“帮我写”。

输入简单提示（如"婉拒明天的会议"），AI 会生成一段草稿。

实测效果：

  • 简短邮件（婉拒/确认/感谢）：质量可接受，微调后可用
  • 复杂邮件（投诉/谈判/提案）：质量不够，建议用专业AI工具
  • 中文邮件：语气偏正式但可以接受，偶尔有翻译腔

结论： Chrome内置AI的优势是离线+隐私+免费。如果你只需要轻度AI辅助（偶尔总结网页、翻译外文、改写句子），内置AI完全够用。如果你是重度AI用户（复杂写作、代码生成、多轮对话），仍需要专业工具。

五、隐私安全须知

Chrome 内置 AI 的隐私保护

Google 官方明确表示：

  1. 网页内容不离开设备：AI在本地处理，不发送网页内容到Google服务器
  2. 选中文本不上传：你选中让AI处理的文本在本地完成所有计算
  3. AI生成结果不回传：AI生成的总结/翻译/改写结果不发送给Google
  4. 无使用日志：AI功能不记录使用历史

需要注意的区别

Chrome 中有两个不同的 AI 功能：

  • Gemini Nano（内置AI）：离线、本地、隐私安全——就是本文介绍的功能
  • Gemini 侧边栏：在线、云端、数据经过Google——这是另一个功能

在 chrome://settings/ai 中可以分别控制这两个功能的开关。如果你担心隐私，可以关闭Gemini侧边栏但保留内置Nano。

模型文件位置

Gemini Nano 模型下载后存储在：

  • Windows: %LOCALAPPDATA%\Google\Chrome\User Data\Default\OptimizationGuide
  • macOS: ~/Library/Application Support/Google/Chrome/Default/OptimizationGuide

如果不再使用AI功能，可以删除该目录释放约1.5GB空间。

六、常见问题与故障排查

问题 解决方案
右键没有AI选项 确认chrome://flags中AI选项已Enabled，重启Chrome
首次使用一直下载中 检查chrome://components页面，确保模型下载完成
AI响应很慢 检查设备内存，关闭其他占内存的标签页
中文输出有翻译腔 尝试在改写时指定"中文母语风格"
AI功能突然消失 Chrome更新后flags可能重置，重新检查chrome://flags
低配设备不可用 内存不足4GB的设备不支持，考虑升级硬件

Chrome 内置 AI 不是要替代 ChatGPT。它的定位是"浏览器里的随手AI助手"——你不需要切换到另一个网站，不需要打开另一个应用，选中文字右键就能用。在日常浏览中，这种"零摩擦"的体验比功能强大但需要切换上下文的工具更实用。
